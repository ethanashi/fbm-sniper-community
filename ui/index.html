📖 How to use Global Spot Radar

What is the Spot Radar?

A high-frequency scanner that monitors the "Spot" markets (direct asset trading) for price gaps between different exchanges or pairs. Unlike P2P, these trades happen instantly on the order book.

Mode 1: Spatial (Inter-Exchange)

Detect price differences for the same coin between Binance and Bybit. You buy on the cheaper exchange and sell on the more expensive one.

ROI = ((Best Bid B - Best Ask A) / Best Ask A) - (Fees)

Mode 2: Triangular (Single Exchange)

Exploits "loops" within a single exchange. For example, buying BTC with USDT, then ETH with BTC, then selling ETH back for USDT. If the cycle ends with more USDT than you started, it's an arbitrage loop.

Tactical Tools

Use the 📋 clipboard icons to copy raw prices instantly. Direct trade links open the specific pair on the exchange site.
